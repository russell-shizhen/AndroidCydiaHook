# AndroidCydiaHook

## Project setup. 

Please open the local.properties file and modify the ndk and sdk directories.

E.g. on mac, it may be like this:
~~~~
sdk.dir=/Users/myname/Library/Android/sdk

ndk.dir=/Users/myname/Library/Android/android-ndk-r15c
~~~~
